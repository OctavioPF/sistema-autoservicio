# Tienda Autoservicio

Proyecto frontend que simula una tienda de autoservicio.

## Funcionalidades
- Render dinámico de productos
- Carrito de compras
- Suma automática del total
- Eliminación de productos
- Persistencia con localStorage

## Tecnologías
- HTML5
- CSS3 (Flexbox / Grid)
- JavaScript Vanilla
- Git & GitHub

## Responsive
Adaptado para dispositivos móviles.
